Working with Modules

Guides / Modules

SynthEdit's modular canvas is built around connectable modules — oscillators, filters, envelopes, effects and dozens of utilities. You assemble a synth by dragging modules onto the canvas and connecting them with virtual patch cables.

Adding a module

Right-click on the canvas to open the module browser, or press Cmd+M on macOS / Ctrl+M on Windows. Modules are organised by category in the left-hand panel. 

// Example: a simple oscillator + filter chain
module "OSC_A" {
  type:  "Oscillator HD";
  pitch: 440;
  shape: "saw";
}

module "FILTER_1" {
  type:    "State Variable Filter";
  cutoff:  2000;
  reso:    0.4;
}
Note

Polyphonic modules require an explicit Voice Combiner before they can drive monophonic outputs like the VCA or the audio output.

Module categories

  • Oscillators & sound sources
  • Filters (LP, HP, BP, SVF, comb)
  • Envelopes & dynamics (ADSR, AHDSR, slew)
  • Effects (reverb, delay, distortion, chorus)
