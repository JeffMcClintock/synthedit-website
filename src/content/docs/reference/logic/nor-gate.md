title: NOR Gate

The **NOR Gate** outputs 5V only when **all** inputs are low. If any input is high, the output is 0V. It is the inverse of the OR gate.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Input 1–N | Audio | Logic inputs |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Output | Audio | Logic result (0V or 5V) |
