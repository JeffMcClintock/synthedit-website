---
title: Binary Counter
description: Steps through a binary count sequence on each clock pulse.
---

The **Binary Counter** increments a binary count on each clock pulse, providing 8 individual binary outputs.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Clock | Audio | Advances count on rising edge |
| Reset | Audio | Resets count to zero |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| B0–B7 | Audio | 8 binary outputs (0V or 5V) |
