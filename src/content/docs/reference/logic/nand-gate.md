---
title: NAND Gate
description: Output goes high when one or more inputs are low.
---

The **NAND Gate** outputs 5V when **any** input is low. Output is 0V only when all inputs are high. It is the inverse of the AND gate.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Input 1–N | Audio | Logic inputs |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Output | Audio | Logic result (0V or 5V) |
