---
title: XOR Gate
description: Output goes high when exactly one input is high.
---

The **XOR Gate** (Exclusive OR) outputs 5V when **exactly one** input is high. If both inputs are high or both are low, the output is 0V.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Input 1–N | Audio | Logic inputs |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Output | Audio | Logic result (0V or 5V) |
