The **Rectifier** performs half-wave rectification, converting negative voltages to positive. This adds harmonics to audio signals and can be used as part of envelope detection circuits.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal In | Audio | Audio input |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal Out | Audio | Rectified output (all positive) |
