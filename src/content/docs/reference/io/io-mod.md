The **IO Mod** provides virtual connections between modules inside a container and modules outside it. When you connect a patch cord across a container boundary, an IO Mod is automatically created.

## Usage Notes

- IO Mods are virtual — they use no CPU
- They appear as plugs on the container's exterior
- Automatically added when you make connections across container boundaries
- You can also insert them manually to create container inputs and outputs
