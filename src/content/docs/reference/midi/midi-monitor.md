The **MIDI Monitor** displays incoming MIDI data in a readable format, useful for debugging MIDI connections and verifying data flow.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| MIDI In | MIDI | MIDI data to display |
| Channel | Control | Filter to a specific MIDI channel |
