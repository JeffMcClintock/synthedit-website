The **MIDI Out** module sends MIDI data to a physical MIDI output port on your computer.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| MIDI In | MIDI | MIDI data to send |

## Usage Notes

- Only functions in the SynthEdit editor — inactive when running as a VST plugin
- Useful for controlling external MIDI hardware (synthesizers, drum machines, effects)
