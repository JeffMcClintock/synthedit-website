---
title: LED Indicator
description: Visual LED light with threshold trigger.
---

The **LED Indicator** displays a visual LED that lights up when the input voltage exceeds a threshold.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal In | Audio | Signal to monitor |
| Threshold | Audio | Voltage threshold for LED activation |

## Parameters

| Property | Description |
|----------|-------------|
| Appearance | LED visual style |
| Hint | Tooltip text |
