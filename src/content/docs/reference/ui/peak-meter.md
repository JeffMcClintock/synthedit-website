---
title: Peak Meter
description: LED bar graph volume meter calibrated in decibels.
---

The **Peak Meter** displays signal level as an LED bar graph, calibrated in decibels.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal In | Audio | Audio signal to meter |
| Response | List | Linear or Decibel scale |
