---
title: Panel Group
description: Visual border for organizing panel controls into sections.
---

The **Panel Group** adds a labeled border around a group of controls on the panel, helping organize the interface into logical sections (e.g., "Oscillator", "Filter", "Envelope").

## Usage

- Only available in Panel View
- Drag to position and resize to encompass the controls you want to group
- Useful for visually separating synth sections on complex panels
