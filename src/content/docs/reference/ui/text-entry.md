The **Text Entry** module provides a text input field for entering text strings, such as filenames for the Wave Player or other text-type connections.

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Text Out | Text | Entered text value |

## Parameters

| Property | Description |
|----------|-------------|
| MIDI Controller ID | Assign a MIDI CC |
| Ignore Program Change | Prevent preset changes |
| Show Title On Panel | Display the label on the panel |
| Hint | Tooltip text |
