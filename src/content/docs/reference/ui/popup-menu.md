The **Popup Menu** provides a pop-up menu for selecting from a list of options.

## Pins

| Pin | Type | Description |
|-----|------|-------------|
| Choice | Control | Currently selected item index |
| Item List | Text | Comma-separated list of menu items |
| Style | Control | Visual style |
| Writeable | Control | Enable/disable editing |
| Greyed | Control | Grey out the control |
| Hint | Text | Tooltip |
| Menu Items | Text | Additional context menu items |
| Heading | Text | Menu heading text |
| Momentary | Control | Resets to zero after selection (useful for command triggers) |
