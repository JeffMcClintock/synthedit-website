The **Scope** displays audio or control voltage signals graphically, similar to an oscilloscope. It supports dual-trace display.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal A | Audio | First trace (displayed in green) |
| Signal B | Audio | Second trace (displayed in yellow) |

## Parameters

| Property | Description |
|----------|-------------|
| Hint | Tooltip text |

## Usage

The module is resizable — drag its edges to make the display larger. Useful for visualizing waveforms, envelopes, LFO shapes, and debugging signal flow.
