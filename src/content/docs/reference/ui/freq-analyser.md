---
title: Frequency Analyser
description: Displays the harmonic spectrum of an audio signal.
---

The **Frequency Analyser** shows the frequency content of an audio signal as a spectrum display.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal In | Audio | Audio signal to analyze |

## Parameters

| Property | Description |
|----------|-------------|
| Frequency Scale | Linear or Logarithmic display |
| Hint | Tooltip text |

## Usage

The module is resizable. Use logarithmic scale for a more musically useful display where each octave takes equal horizontal space.
