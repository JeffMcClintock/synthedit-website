---
title: Fixed Values
description: Provides constant voltage sources.
---

The **Fixed Values** module provides one or more constant voltage outputs. Set the voltage in the module's properties dialog.

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Value 1–N | Control | Constant voltage outputs (new outputs created as needed) |

## Usage

Connect to any input that needs a fixed value. A new "Spare Value" output is always available — connecting to it creates the output and adds another spare.
