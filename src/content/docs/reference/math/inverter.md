---
title: Inverter
description: Negates the input signal.
---

The **Inverter** flips the polarity of the input signal (multiplies by -1). A 5V input becomes -5V, and vice versa.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal In | Audio | Signal to invert |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal Out | Audio | Inverted signal |
