The **Divide** module outputs the quotient of two input signals.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Input 1 | Audio | Numerator signal |
| Input 2 | Audio | Denominator signal |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Output | Audio | Input 1 ÷ Input 2 |
