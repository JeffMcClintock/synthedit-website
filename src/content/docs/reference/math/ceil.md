---
title: Ceil (Ceiling)
description: Rounds up to the next whole voltage value.
---

The **Ceil** module rounds the input voltage up to the next whole number (e.g., 4.2V becomes 5.0V).

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal In | Audio | Signal to round |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Signal Out | Audio | Rounded-up output |
