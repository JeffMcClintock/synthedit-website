---
title: Subtract
description: Subtracts one signal from another.
---

The **Subtract** module outputs the difference of two input signals.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Input 1 | Audio | First signal |
| Input 2 | Audio | Signal to subtract |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Output | Audio | Input 1 − Input 2 |
