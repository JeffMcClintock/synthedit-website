---
title: Multiply
description: Multiplies two signals together.
---

The **Multiply** module outputs the product of two input signals.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Input 1 | Audio | First signal |
| Input 2 | Audio | Second signal |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Output | Audio | Input 1 × Input 2 |
