---
title: Switch (Many to 1)
description: Selects one of multiple inputs to route to a single output.
---

The **Switch (Many to 1)** selects one of several input signals and routes it to a single output.

## Inputs

| Pin | Type | Description |
|-----|------|-------------|
| Choice | Control | Selects which input is routed to the output |
| Input 1–N | Audio | Multiple inputs (new inputs auto-created when connected) |

## Outputs

| Pin | Type | Description |
|-----|------|-------------|
| Output | Audio | Selected input signal |

## Usage

New input pins are automatically created as you connect to them. Only the selected input is heard at the output.
