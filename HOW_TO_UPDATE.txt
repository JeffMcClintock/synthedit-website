HOW TO UPDATE THE SYNTHEDIT WEBSITE
===================================

LOCAL DEVELOPMENT
-----------------
1. Make changes to files in src/content/docs/ (markdown files)
2. Test locally:
   npm run dev
   (opens at localhost:4321/new/)
3. Check your changes in the browser

DEPLOYING TO SYNTHEDIT.COM/NEW/
--------------------------------
1. Build the site:
   npm run build

2. Upload the contents of the dist/ folder to your server at /new/
   (All files and folders inside dist/ go into the /new/ directory)

3. That's it! The changes will be live at www.synthedit.com/new/

COMMON UPDATES
--------------
- Add/edit a guide page: Create/edit .md file in src/content/docs/guides/
- Add/edit module reference: Create/edit .md file in src/content/docs/reference/
- Update images: Place in public/images/ and reference as /images/filename.jpg
- Update changelog: Edit src/content/docs/guides/installation.mdx

BEFORE GOING LIVE
-----------------
When ready to move to www.synthedit.com (root):
1. Remove base: '/new/' from astro.config.mjs
2. Rebuild: npm run build
3. Upload dist/ contents to the root directory
